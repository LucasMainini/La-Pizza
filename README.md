# La Pizza

Site institucional e cardápio online para uma pizzaria, com foco em apresentação dos produtos e contato via WhatsApp.

## Recursos
- Layout hero com imagem e texto
- Cardápio com abas (salgadas e doces)
- Animações de entrada no scroll
- Botão com efeito de ondinha
- Formulário de contato com envio para WhatsApp
- Responsivo para celular

## Como executar
1. Abra o arquivo `index.html` no navegador.
2. Para editar estilos, use `styles/styles.css` e `styles/header.css`.

## Estrutura
- `index.html` — página principal
- `styles/styles.css` — estilos gerais
- `styles/header.css` — estilos do topo
- `script.js` — interações (menu mobile e animações)
- `img/` — imagens do site

## Observações
- Para o WhatsApp funcionar, adicione seu número em `index.html` na variável `telefone`.
